### ✨ 主な機能

<div style="display: grid; grid-template-columns: repeat(auto-fit, minmax(280px, 1fr)); gap: 1rem; margin: 1.5rem 0;">

  <div style="padding: 1.25rem; background: rgba(10, 6, 20, 0.4); backdrop-filter: blur(10px); border-radius: 12px; border-left: 4px solid #a855f7;">

    <h4 style="margin: 0 0 .5rem 0; color: #a855f7;">アイテム作成</h4>
    <p style="margin: 0; color: rgba(255,255,255,0.7); font-size: .9rem;">紙1枚でアイテム作成。タイトル、内容、画像を追加し、開始日/有効期限を設定できます。</p>
  </div>

  <div style="padding: 1.25rem; background: rgba(10, 6, 20, 0.4); backdrop-filter: blur(10px); border-radius: 12px; border-left: 4px solid #14b8a6;">

    <p style="margin: 0; color: rgba(255,255,255,0.7); font-size: .9rem;">固有のドメイン名を刻印してアプリ内の身元として使用。刻印したドメイン名がすべての活動に表示されます。</p>
  </div>

  <div style="padding: 1.25rem; background: rgba(10, 6, 20, 0.4); backdrop-filter: blur(10px); border-radius: 12px; border-left: 4px solid #d946ef;">
    <div style="font-size: 2rem; margin-bottom: .5rem;">🔖</div>
    <h4 style="margin: 0 0 .5rem 0; color: #d946ef;">スタンプ & ログ</h4>
    <p style="margin: 0; color: rgba(255,255,255,0.7); font-size: .9rem;">スタンプでテンプレート作成。単独ログ（個人用）または共有ログ（コミュニティ）モードで記録を残せます。</p>
  </div>

  <div style="padding: 1.25rem; background: rgba(10, 6, 20, 0.4); backdrop-filter: blur(10px); border-radius: 12px; border-left: 4px solid #a855f7;">
    <div style="font-size: 2rem; margin-bottom: .5rem;">📤</div>
    <h4 style="margin: 0 0 .5rem 0; color: #a855f7;">QRコード転送</h4>
    <p style="margin: 0; color: rgba(255,255,255,0.7); font-size: .9rem;">相手のQRをスキャン → アイテム選択 → 転送完了！所有権が即時移転されます。</p>
  </div>

  <div style="padding: 1.25rem; background: rgba(10, 6, 20, 0.4); backdrop-filter: blur(10px); border-radius: 12px; border-left: 4px solid #14b8a6;">

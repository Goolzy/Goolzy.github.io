#### ショップアイテムレイアウト改善
モバイル画面でショップアイテムがより見やすくなるようレイアウトが改善されました。
- **3行構造**: アイテム情報と価格が分離され、狭い画面でも情報が切れずに表示
- **価格右寄せ**: 価格情報が別の行に右寄せで整然と配置
- **割引タグの十分なスペース**: 割引バッジが付いてもアイテム名と説明が完全に表示

#### 拡張可能なプロフィールヘッダー
ホーム画面上部のプロフィール領域がドロワースタイルのパネルに改善されました。
- **ドラッグで展開**: ハンドルバーを下にドラッグするか、ユーザー名をタップしてプロフィールパネルを展開
- **統合パネル**: ヘッダーとプロフィールコンテンツ（プロフィールカード、QRコード、API情報）が1つのパネルに統合
- **スムーズなアニメーション**: 展開/折りたたみ時の自然なアニメーション効果
- **固定されたメインコンテンツ**: プロフィール展開時もインベントリグリッドは所定の位置に維持

#### ポケット リブランディング
アイテムを分類する単位の名前が「パーティション」から「ポケット」に変更されました。
- **直感的な名前**: 物を入れるポケットのように、アイテムを収納する空間という意味をより適切に伝達
- **一貫したUI**: アプリ全体で「ポケット」という用語に統一
- **多言語サポート**: 韓国語(포켓)、英語(Pocket)、日本語(ポケット)、中国語(口袋)など各言語に適切に翻訳

#### 既存データの自動マイグレーション
既存の「パーティション」データが「ポケット」に自動変換されます。
- **ロスレスマイグレーション**: 既存のパーティションとアイテムはそのまま維持
- **自動処理**: アプリ更新時に自動的にマイグレーションが実行
- **シームレスな移行**: ユーザーが何か操作する必要なく、そのまま使用可能

#### アクションスピードダイヤルFAB
アイテム選択時に使用可能なアクションがフローティングボタンで表示されます。
- **フローティングアクションボタン**: アイテム選択時に右下に稲妻マークのFABが表示
- **スピードダイヤル**: FABをタップすると上に展開して利用可能なアクションボタンを表示
- **アクション数バッジ**: FABに現在使用可能なアクション数がバッジで表示
- **スムーズなアニメーション**: FABの表示/非表示スケールアニメーション、アクション数変更時のポップエフェクト

#### アイテム選択UIの改善
選択したアイテムの視覚的フィードバックが改善されました。
- **外側アウトライン**: 選択枠が画像の外側に表示され、元のサイズを維持
- **メインカラーボーダー**: 選択アウトラインがテーマのメインカラーで統一
- **重なるボーダー**: 隣接するアイテムを選択すると、枠が重なって選択範囲を明確に表示
- **揺れ効果の削除**: 選択されたアイテムの揺れ効果が削除され、よりすっきりしたUI

#### ログ画面の未読通知の改善
ログリストで未読状態をより明確に確認できます。
- **未読カウントバッジ**: 未読コメント数がバッジとして表示（最大99+）
- **既読/未読の色分け**: 未読ログは明るい背景、既読ログは暗い背景で視覚的に区別
- **リアルタイム更新**: 新しいコメントが投稿されるとリアルタイムでカウント増加、確認後自動リセット

#### メールボックスショートカット
ボトムナビゲーションバーからメールボックス（受信画面）に直接移動できます。
- **クイックアクセス**: ホーム画面から直接受信アイテムを確認可能
- **直感的なアイコン**: メールアイコンでメールボックス機能を表現

#### デフォルトポケット
最初のポケットがヘッダーなしでデフォルトポケットとして機能します。
- **ヘッダー非表示**: デフォルトポケットはポケットヘッダーが表示されず、よりスッキリしたUI
- **常に最上部**: デフォルトポケットは常にインベントリの最上部に位置
- **折りたたみ不可**: デフォルトポケットは常に展開された状態を維持
- **自動アイテム受信**: 新しく購入または受け取ったアイテムはデフォルトポケットに自動的に移動

#### ショップ画面UIの簡素化
ショップ画面のヘッダー領域が整理されました。
- **重複タイトルの削除**: シェルモードでナビバータイトルと重複していた「ショップ」タイトルを削除
- **説明文の削除**: 不要な説明文を削除し、商品リストに集中
- **お知らせを上部に配置**: Firebaseから取得するお知らせメッセージがタブバーの上部に表示

#### インベントリホームのオープニングアニメーション
インベントリホーム画面にも他のタブと同じオープニングアニメーションが適用されます。
- **ヘッダースライドイン**: アプリ起動時にプロファイルヘッダーが上から滑らかに降りてきます
- **タブ切り替えアニメーション**: 他のタブからホームに戻る際も同じアニメーションが適用
- **一貫したUX**: すべてのタブで同じオープニング演出による統一感のあるユーザー体験

#### 設定画面の改編
設定画面が5つの折りたたみグループに再構成されました。
- **5つのグループ**: 色、フォント、サウンド、プッシュ通知、システムに設定項目を分類
- **折りたたみ/展開**: 各グループをタップして折りたたみ/展開できるアコーディオンUI
- **プッシュ通知の簡素化**: 7カテゴリが3つ（取引、ログ、期限）に統合
- **キーボードアクセシビリティ**: 方向キーでグループの折りたたみ/展開、項目移動をサポート

#### バグ修正
- **通知パネル表示の修正**: プロファイルヘッダーの通知パネルで通知が正しく表示されるようになりました。
  - 通知タイプ別のアイコンと色が正確に表示されます
  - 通知のタイトルと内容が多言語で正しく表示されます
  - 通知アイテムの角丸がメニューボタンと同じスタイルに統一されました
- **ショップヘッダーの重複修正**: 特定の状況でショップ画面にヘッダーが2つ表示される問題が修正されました。
  - ホーム画面に復帰後、ショップタブ選択時のシェルモード状態の不一致を解決
  - 参照カウント方式でシェルモード状態の安定した管理を保証
- **ポケット折りたたみ/展開アニメーション改善**: ポケットを折りたたむ際にコンテンツが瞬間的に消えていた問題を修正し、スムーズに折りたたまれるアニメーションに改善しました。
  - 折りたたみ: コンテンツが下から上へ自然に折りたたまれます
  - 展開: コンテンツが上から下へスムーズに展開されます

#### 共有テンプレート検索の改善
共有テンプレート画面の検索機能が改善されました。
- **即時閲覧**: 画面に入ると最近共有されたテンプレートがすぐに表示されます（検索不要）
- **メニュー名変更**: 「共有テンプレート」→「共有テンプレート検索」に変更し、より明確な案内
- **高速読み込み**: 3つのAPI呼び出しを1つに統合し、画面の読み込み速度を大幅に改善
- **タグテーマカラー**: 人気/最新タグが固定色の代わりにアプリテーマカラーで表示

#### 共有テンプレート詳細画面のデザイン改善
共有テンプレートの詳細画面がニューモーフィックデザインに改善されました。
- **ニューモーフィックデザイン**: アプリ全体と一貫したニューモーフィックスタイルに統一
- **キーワード2列表示**: キーワードが「ラベル:値」形式で2列レイアウトに整理され、可読性向上
- **タグデザイン改善**: タグチップがニューモーフィックスタイルに変更され、視覚的統一感

#### ログナビゲーション統合
ログでクリップアイテムをタップすると、グリッドと同様にクリップ詳細画面に移動します。
- **クリップ詳細画面への移動**: ログでクリップアイテムをタップすると、クリップ内の全ページをスワイプできる詳細画面に移動
- **正確なページ位置**: タップしたアイテムのクリップ内の順序に対応するページから開始
- **統一されたパス**: グリッド、ログ、キーボードナビゲーション全てが同じ詳細画面経路を使用

#### グリッドアイテムカードデザイン改善
インベントリグリッドのアイテムカードがよりクリーンなフラットデザインに変更されました。
- **シャドウ削除**: 密集したグリッドでインセットシャドウのように見えたBoxShadowを完全削除
- **下部グラデーション削除**: カード下部の暗いグラデーションオーバーレイを削除
- **フラットデザイン**: クリーンでモダンなフラットスタイルに統一
- **テキストラベル維持**: アイテムタイプラベルは白いテキスト+シャドウで可読性を確保

#### ウィザードチャットルーム作成の改善
チャットルームウィザードがより直感的に改善されました。
- **グループ会話人数表示**: デフォルトタイトルが「n人グループ会話」と表示され、参加者数をすぐに確認
- **完成イメージプレビュー**: ウィザードメニューにアイコンの代わりに、実際に完成するアイテムイメージが表示
- **高速イメージ読み込み**: ローカルイメージ使用により、毎回ダウンロードせずに即座に表示

#### 1:1チャットルーム動的タイトル
1:1チャットルームで送信者と受信者が相手の名前をタイトルとして表示できます。
- **相手の名前表示**: 送信者は「受信者様との会話」、受信者は「送信者様との会話」と表示
- **カスタムタイトル優先**: 直接入力したタイトルは両者に同じように表示
- **グループチャット同一タイトル**: グループチャットはすべての参加者に同じタイトルを表示

#### バッジ画像読み込みの改善
アプリ再起動時にバッジ画像がちらつく問題が解決されました。
- **永続キャッシュ**: バッジ画像URLがローカルに保存され、アプリ再起動後も維持
- **プリロード**: スプラッシュ画面ですべてのキャッシュされたバッジ画像を事前読み込み
- **バックグラウンド更新**: サーバーでバッジ画像が変更されると、バックグラウンドで自動更新

#### 絵文字システムの改善
ログに絵文字を添付する機能が改善されました。
- **キャッシュ復元修正**: 絵文字送信後、再入場しても正常に表示
- **ボックスサイズ最適化**: 絵文字のみのメッセージの吹き出しサイズが適切に調整
- **選択プレビュー**: 絵文字選択時、入力欄の上に選択した絵文字のプレビューを表示

#### チャット入力欄のフォーカス改善
iPadでの入力欄フォーカス問題が修正されました。
- **iPad互換性**: キーボード表示時、入力欄に正常にフォーカスが設定
- **安定した入力**: すべてのiOSデバイスで一貫した入力体験を提供

#### 通知フィルタリングの改善
現在表示中の画面に対する通知がフィルタリングされます。
- **プッシュ通知フィルター**: 現在開いているアイテムのプッシュ通知が表示されない
- **アプリ内通知フィルター**: 現在表示中のアイテムのアプリ内通知トーストが表示されない

#### 通知トーストジェスチャー
アプリ内通知トーストにジェスチャーで操作できるようになりました。
- **タップで移動**: 通知をタップすると、該当ドキュメントへ直接移動します
- **スワイプで閉じる**: 上にスワイプすると、通知をすばやく閉じることができます
- **触覚フィードバック**: タッチ操作に触覚フィードバックが提供されます

#### バグ修正
- 絵文字キャッシュ復元時のemoticonId欠落問題を修正
- Cloud Functionsアプリ内通知作成エラーを修正

#### フォントサイズ調整
設定画面からアプリ全体のフォントサイズを3段階で調整できます。
- **3段階選択**: 標準(1.0x)、中(1.3x)、大(1.5x)から選択
- **即時適用**: 設定変更がアプリ全体に即座に反映
- **設定保持**: アプリを再起動しても設定が維持されます

#### キーボード自動非表示
アイテム詳細画面でログ入力欄以外の領域をタップすると、キーボードが自動的に閉じます。
- **空白部分タップ**: 入力欄以外の領域をタッチするとキーボードが自動的に閉じる
- **iOSサポート**: iPhoneでキーボード閉じるボタンがなくても快適に使用可能

#### バックグラウンド音楽の維持
アプリ実行時にデバイスで再生中だった音楽（Spotify、YouTube Music、Apple Musicなど）が中断されなくなりました。
- **音楽の連続再生**: アプリを起動しても他のアプリのバックグラウンド音楽が引き続き再生される
- **効果音のミキシング**: アプリの効果音がバックグラウンド音楽と一緒に再生される
- **マナーモード連動**: マナーモードではアプリの効果音も消音される

#### 検索履歴とフィルター機能の改善
スーパー検索バーが検索履歴を保存し、フィルター機能が改善されました。
- **検索履歴の保存**: フィルターの追加や共有テンプレートの検索時に検索語が自動保存される
- **最近の検索表示**: 検索バーにフォーカスすると最近の検索5件が表示される
- **クイックフィルター追加**: 最近の検索をタップするとすぐにフィルターとして追加される
- **ボタンUIの改善**: フィルター追加と共有テンプレートボタンが1行にコンパクトに配置される

#### ポケット管理の改善
新しいニューモーフィックデザインのポケット名設定ダイアログとドラッグ並べ替え機能を追加しました。
- **ニューモーフィックダイアログ**: ダーク/ライトモード対応の美しい新デザイン
- **ドラッグ並べ替え**: ハンドルアイコンを長押ししてポケットをドラッグして並べ替え
- **作成時に名前設定**: 新しいポケット作成時にカスタム名を設定可能

#### リンクプレビュー説明表示
ログに添付されたリンクカードにWebページの説明(OpenGraph description)が表示されます。
- **説明自動抽出**: Webページのog:descriptionメタタグから説明を抽出
- **カードUI改善**: タイトル下に説明が最大2行で表示
- **対応プラットフォーム**: YouTube、Spotify、一般Webページなど、すべてのリンクタイプに対応

#### テキストスタイルマーカー変更
メインカラーテキストマーカーが`<テキスト>`から`{テキスト}`に変更されました。
- **引用符の競合解決**: `>` 文字を使用する引用機能との競合を防止
- **一貫したマーカーシステム**: 括弧類マーカー(`{}`、`[]`、`()`)に統一

#### 複数引用文のサポート
ログで複数の引用文を同時に表示できます。
- **複数引用のレンダリング**: 2つ以上の`>` 引用文が各々個別の引用ウィジェットとして表示
- **順序の保持**: 入力した順序のままで引用文が表示

#### メール送信検証の強化
送信機能でメールアドレスの有効性をより正確に検証します。
- **MXレコード検証**: 受信者メールドメインのメールサーバー存在確認
- **エラー防止**: 存在しないドメインへの送信試行を事前に遮断

#### ドラッグ操作の改善
選択したアイテムを長押しせずにすぐドラッグして移動できます。
- **即時ドラッグ**: アイテム選択後すぐにドラッグ可能（長押し不要）
- **揺れアニメーション**: 選択されたアイテムが微細に揺れてドラッグ可能状態を表示
- **直感的な操作**: iOSホーム画面編集モードに似た親しみやすいUX

#### Dropboxファイル添付
Dropboxを接続してログにファイルを添付できます。
- **多様なファイル対応**: 画像、動画、PDF、音楽、テキストファイルの添付が可能
- **共有ログ対応**: 共有されたアイテムのログでも添付ファイルの閲覧・ダウンロード可能
- **内蔵ビューア**: アプリ内でファイルを直接確認（画像拡大、PDFページめくり、動画再生など）

function copyVersionContent(button, event) {
  event.stopPropagation();
  event.preventDefault();

  const details = button.closest('details.changelog-version');
  const versionContent = details.querySelector('.version-content');
  const versionTitle = details.querySelector('.version-title');

  const titleText = versionTitle ? versionTitle.textContent.trim() : '';
  const contentClone = versionContent.cloneNode(true);

  contentClone.querySelectorAll('h4').forEach(h4 => {
    h4.textContent = '\n## ' + h4.textContent.trim() + '\n';
  });
  contentClone.querySelectorAll('li').forEach(li => {
    li.textContent = '- ' + li.textContent.trim();
  });

  const text = '# ' + titleText + '\n' + contentClone.textContent
    .replace(/\n{3,}/g, '\n\n').trim();

  navigator.clipboard.writeText(text).then(() => {
    const copyIcon = button.querySelector('.copy-icon');
    const checkIcon = button.querySelector('.check-icon');
    copyIcon.style.display = 'none';
    checkIcon.style.display = 'block';
    button.classList.add('copied');
    setTimeout(() => {
      copyIcon.style.display = 'block';
      checkIcon.style.display = 'none';
      button.classList.remove('copied');
    }, 2000);
  });
}

#### Расширяемый заголовок профиля
Область профиля в верхней части главного экрана была переработана в виде выдвижной панели.
- **Потяните для расширения**: Потяните ручку вниз или нажмите на имя пользователя, чтобы развернуть панель профиля
- **Единая панель**: Заголовок и содержимое профиля (карточка профиля, QR-код, информация API) объединены в одной панели
- **Плавная анимация**: Естественные эффекты анимации при разворачивании/сворачивании
- **Фиксированный основной контент**: Сетка инвентаря остается на месте при развернутом профиле

#### Ребрендинг Pocket
Название единиц классификации предметов изменено с 'Partition' на 'Pocket'.
- **Интуитивное название**: Лучше передает концепцию пространства для хранения предметов, как карман
- **Согласованный UI**: Унифицированная терминология 'Pocket' во всем приложении
- **Многоязычная поддержка**: Соответствующий перевод для каждого языка

#### Автоматическая миграция данных
Существующие данные 'Partition' автоматически преобразуются в 'Pocket'.
- **Миграция без потерь**: Ваши существующие разделы и предметы сохраняются в точности
- **Автоматическая обработка**: Миграция происходит автоматически при обновлении приложения
- **Бесшовный переход**: Никаких действий не требуется - просто обновите и продолжайте использовать

#### Адаптивная раскладка панели реакций
Панель реакций изображения в детальном просмотре элемента правильно отображается на экранах всех размеров.
- **Адаптивная раскладка**: Кнопки реакции равномерно распределены даже на узких экранах
- **Полная ширина**: Панель использует всю ширину экрана независимо от размера изображения
- **Компактный дизайн**: Оптимизированные отступы уменьшают закрытие фотографии при сохранении площади касания

#### Бесконечная прокрутка поиска общих шаблонов
Когда результатов поиска общих шаблонов более 20, вы можете прокрутить для загрузки дополнительных результатов.
- **Бесконечная прокрутка**: Автоматически загружает следующие 20 результатов при достижении конца списка
- **Индикатор загрузки**: Отображается индикатор прогресса внизу во время загрузки дополнительных результатов
- **Обратная совместимость**: Предыдущие версии приложения по-прежнему отображают 20 результатов

#### Обновления домашней страницы
- **Поддержка китайского (упрощённого) языка**: Добавлена локализация 中文(简体) на главную страницу (8-й язык)
- **Отображение поддерживаемых языков**: Ниже значков загрузки платформ отображаются 8 флагов языков

#### Параметры языка App Store
- **Добавлены поддерживаемые языки iOS/macOS**: Добавлен китайский язык (zh) в CFBundleLocalizations

#### Улучшение выбора даты
- **Расширение диапазона выбора**: В интерфейсе выбора даты теперь можно выбрать год с 1900 года (ранее было 2000 год)

#### Функция обмена документами
Содержимое документов можно скопировать в буфер обмена или отправить по электронной почте из экрана деталей.
- **Копирование в буфер обмена**: Копирование в виде чистого текста с удалением маркеров форматирования
- **Отправка по электронной почте**: Отправка форматированного содержимого в формате HTML
- **Выборочное включение**: Выбор нужных элементов из названия, содержимого, ключевых слов, обсуждения и системного журнала
- **Поддержка режима клипа**: Функция работает одинаково в экране просмотра деталей клипа

#### Синхронизация блокировки клипа
Статус блокировки элементов в клипе синхронизируется.
- **Массовая блокировка/разблокировка**: При блокировке основного элемента клипа все вложенные элементы также блокируются
- **Синхронизация пароля**: Все элементы в клипе используют один и тот же пароль блокировки

#### Автоматическая прокрутка при перетаскивании
При перетаскивании элементов экран автоматически прокручивается на краях.
- **Зона автоматической прокрутки**: Перетащите в верхнюю/нижнюю 10% экрана для автоматической прокрутки
- **Прокрутка на основе ускорения**: Скорость прокрутки увеличивается по мере приближения к краю
- **Тактильная обратная связь**: Легкая вибрация при входе в зону прокрутки
- **Навигация между разделами**: Быстро просматривайте множество элементов или несколько разделов

#### Навигация при прокрутке
Быстро переходите в начало или конец экрана сведений.
- **Кнопки навигации**: Кнопки прокрутки появляются рядом со строкой поиска при наличии большого количества комментариев
- **Отображение на основе позиции**: Только нижняя кнопка видна в начале, только верхняя кнопка видна в конце
- **Дизайн Neumorphic**: Стиль кнопок соответствует дизайну приложения

#### Восстановлена анимация перехода между страницами
Анимация переворота для нижней панели навигации и экрана получения восстановлена.
- **3D эффект переворота**: Переходы между страницами с поворотом на 180 градусов по оси Y
- **Страницы журнала/магазина**: Применена анимация переворота
- **Экран получения**: Применена анимация переворота из кнопки почтового ящика и ссылок глубокого связывания уведомлений

#### Объединение дизайна интерфейса
Дизайн Neumorphic был применён к экранам отправки, получения и магазина, объединяя весь интерфейс приложения.
- **Экраны отправки/получения**: Тени Neumorphic применены к карточкам и диалогам
- **Экран магазина**: Эффекты свечения на кнопках вкладок, карточки товаров в стиле neumorphic
- **Дизайн поля ввода**: Встроенный стиль чётко выделяет области ввода

#### Анимация переупорядочивания журнала
Когда элементы перемещаются в начало списка журнала из-за новых комментариев, применяются плавные анимации.
- **Эффект масштабирования**: Элементы кратко увеличиваются в размере, затем возвращаются в нормальный размер после переупорядочивания
- **Последовательная обработка**: Несколько элементов анимируются по очереди при одновременном перемещении
- **Визуальный акцент**: Ясно указывает, какие элементы наиболее свежие

#### Стабилизация базы данных
Проблемы совместимости базы данных, которые могли возникнуть при обновлении приложения, были решены.
- **Автоматическое восстановление**: Автоматически воссоздаёт базу данных при несовпадении версии
- **Сохранение данных**: Потеря данных исключена благодаря автоматической синхронизации с сервером
- **Все платформы**: Одинаковое исправление применено к iOS, Android, macOS и Windows

#### Улучшение защиты паролем
Безопасность и удобство использования защищённых паролем элементов значительно улучшены.
- **Размытое перекрытие**: Содержимое полностью скрыто при входе в защищённые элементы для повышенной безопасности
- **Дизайн Neumorphic**: Диалоги установки/разблокировки пароля теперь используют стиль neumorphic
- **Отображение 16 цифр**: Маленькие кружочки (всего 16) показывают статус ввода пароля
- **Двухстрочный ввод**: Установите пароль и подтверждение на одном экране

#### Улучшение настроек push-уведомлений
Экран настроек push-уведомлений был переорганизован по категориям для более удобного управления.
- **7 категорий**: Классифицировано на транзакции, журнал, срок действия, реакции, ключевые слова, общее использование и система
- **Переключатель группы**: Включайте или выключайте всю категорию за один раз
- **Свернуть/развернуть**: Пользовательский интерфейс аккордеона позволяет просматривать только необходимые категории
- **Подробные уведомления**: Добавлены уведомления о принятии/отклонении транзакций, реакциях, ключевых словах, уведомлениях об истечении срока действия общего использования/отчёт и системные уведомления
- **Дизайн Neumorphic**: Стиль neumorphic применён к отдельным элементам уведомлений

#### Улучшение настроек звука
Кнопка отключения звука теперь имеет дизайн neumorphic для интуитивного отображения нажатого состояния.
- **Эффект нажатия**: Кнопка выглядит нажатой внутрь при отключении звука
- **Тень Neumorphic**: Состояние по умолчанию показывает выпуклую тень, нажатое состояние показывает вогнутую тень

#### Улучшение производительности загрузки журнала
Улучшена скорость загрузки общего журнала.
- **Загрузка с приоритетом кэша**: Ранее просмотренные журналы мгновенно отображаются из кэша
- **Экономия данных**: Нет запросов к серверу при повторном посещении, используются локальные данные
- **Новые комментарии в реальном времени**: Новые комментарии получаются в реальном времени даже при использовании кэша

#### Функция редактирования значка
Отредактируйте изображение и ключевые слова гравированных значков.
- **Меню редактирования**: Меню "Редактировать значок" появляется при выборе гравированного значка
- **Изменение изображения**: Замените изображение значка новым
- **Управление ключевыми словами**: Добавляйте, удаляйте или изменяйте 5 типов ключевых слов

#### Оптимизация производительности изображения
Значительно улучшена скорость загрузки сетевого изображения.
- **Локальный кэш**: После загрузки изображения сохраняются локально для мгновенного отображения
- **Синхронизация изображения значка**: Изменения изображения значка немедленно отражаются на всех журналах

#### Автозаполнение недавних получателей
Быстро выбирайте недавних получателей на экране отправки.
- **Список автозаполнения**: Раскрывающееся меню показывает до 30 недавних получателей при фокусировке на поле ввода
- **Быстрый выбор**: Автоматически выполняет поиск при выборе получателя из списка
- **Повторный поиск**: Очистите выбор кнопкой X для поиска другого получателя
- **Локальное хранилище**: Информация о получателе автоматически сохраняется при успешной отправке

#### Улучшение ограничений отправки
Экипированные значки и общие марки теперь запрещены к отправке.
- **Экипированные значки**: Снимите значок для его отправки
- **Общие марки**: Отозовите общее использование для отправки марки
- **Подробное руководство**: Отображает конкретные сообщения, объясняющие причину ограничения

#### Система реакций
Реагируйте на элементы эмоциональными выражениями.
- **12 реакций**: Различные эмоциональные выражения, включая "Нравится", "Люблю", "Лучшее", "Поздравляю" и другие
- **Переключение реакции**: Отмените нажатием на выбранную реакцию снова
- **Регистрация в журнале**: Системный журнал записывает при выборе или отмене реакций
- **Многоязычная поддержка**: Названия реакций отображаются на языке пользователя

#### Улучшение функции поиска
Поиск в реальном времени расширен для включения имён элементов и ключевых слов.
- **Поиск по имени элемента**: Поиск по названию документа, описанию марки или имени домена значка
- **Поиск по ключевым словам**: Поиск по ключевым словам, сохранённым в марках и значках
- **Поиск по тегам**: Поиск по тегам шаблонов марок

#### Улучшение уведомлений в приложении
Уведомления в виде всплывающего окна в реальном времени улучшены.
- **Отображение значка отправителя**: Изображение значка отправителя отображается встроено в уведомление
- **Стиль Dynamic Island**: Элегантная анимация всплывающего окна, вдохновлённая стилем iOS Dynamic Island
- **Обнаружение в реальном времени**: Точно обнаруживает новые и обновленные уведомления на основе ID+временной метки

<script>
function copyVersionContent(button, event) {
  event.stopPropagation();
  event.preventDefault();

  const details = button.closest('details.changelog-version');
  const versionContent = details.querySelector('.version-content');
  const versionTitle = details.querySelector('.version-title');

  const titleText = versionTitle ? versionTitle.textContent.trim() : '';
  const contentClone = versionContent.cloneNode(true);

  contentClone.querySelectorAll('h4').forEach(h4 => {
    h4.textContent = '\n## ' + h4.textContent.trim() + '\n';
  });
  contentClone.querySelectorAll('li').forEach(li => {
    li.textContent = '- ' + li.textContent.trim();
  });

  const text = '# ' + titleText + '\n' + contentClone.textContent
    .replace(/\n{3,}/g, '\n\n').trim();

  navigator.clipboard.writeText(text).then(() => {
    const copyIcon = button.querySelector('.copy-icon');
    const checkIcon = button.querySelector('.check-icon');
    copyIcon.style.display = 'none';
    checkIcon.style.display = 'block';
    button.classList.add('copied');
    setTimeout(() => {
      copyIcon.style.display = 'block';
      checkIcon.style.display = 'none';
      button.classList.remove('copied');
    }, 2000);
  });
}
</script>

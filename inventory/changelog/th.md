#### การเลื่อนไปในการค้นหาแม่แบบที่แชร์ไม่จำกัด
เมื่อมีผลลัพธ์การค้นหาแม่แบบที่แชร์มากกว่า 20 รายการ คุณสามารถเลื่อนเพื่อโหลดเพิ่มเติมได้
- **การเลื่อนไปไม่จำกัด**: อัตโนมัติโหลดผลลัพธ์ 20 รายการถัดไปเมื่อถึงด้านล่าง
- **ตัวบ่งชี้การโหลด**: แสดงตัวบ่งชี้ความคืบหน้าที่ด้านล่างขณะกำลังโหลดเพิ่มเติม
- **ความเข้ากันได้แบบย้อนหลัง**: เวอร์ชันแอปก่อนหน้ายังคงแสดงผลลัพธ์ 20 รายการเหมือนเดิม

#### การอัพเดตหน้าแรก
- **รองรับ Chinese (Simplified)**: เพิ่มการแปลภาษา 中文(简体) ให้กับหน้าแรก (ภาษาที่ 8)
- **การแสดงภาษาที่รองรับ**: แสดงไอคอนธงชาติ 8 ภาษาใต้ไอคอนการดาวน์โหลดแพลตฟอร์ม

#### การตั้งค่าภาษา App Store
- **เพิ่มภาษาที่รองรับ iOS/macOS**: เพิ่มภาษาจีน (zh) ใน CFBundleLocalizations

#### การปรับปรุงการเลือกวันที่
- **ขยายช่วงการเลือก**: เลือกจากปี 1900 ได้ในส่วนอินเทอร์เฟซเลือกวันที่ (ก่อนหน้าคือปี 2000)

#### การเลื่อนอัตโนมัติเมื่อลาก
เมื่อลากรายการ หน้าจอจะเลื่อนอัตโนมัติที่ขอบ
- **โซนการเลื่อนอัตโนมัติ**: ลากไปยังส่วนบนสุด/ล่างสุด 10% ของหน้าจอเพื่อเลื่อนอัตโนมัติ
- **ตามการเร่ง**: ความเร็วในการเลื่อนเพิ่มขึ้นเมื่อคุณเข้าใกล้ขอบมากขึ้น
- **ข้อเสนอแนะแบบ Haptic**: แจ้งเตือนการสั่นเบาเมื่อเข้าสู่โซนการเลื่อน
- **การนำทางข้ามพาร์ติชัน**: เรียกดูรายการหรือพาร์ติชันมากมายได้อย่างรวดเร็ว

#### การนำทางการเลื่อน
เข้าไปที่ด้านบนหรือด้านล่างได้อย่างรวดเร็วในหน้าจอมุมมองรายละเอียด
- **ปุ่มนำทาง**: ปุ่มเลื่อนปรากฏขึ้นข้างแถบค้นหาเมื่อมีความเห็นจำนวนมาก
- **การแสดงตามตำแหน่ง**: แสดงเฉพาะปุ่มด้านล่างที่ด้านบน แสดงเฉพาะปุ่มด้านบนที่ด้านล่าง
- **การออกแบบ Neumorphic**: สไตล์ปุ่มสอดคล้องกับการออกแบบแอป

#### การเลื่อนเข้าชั้นแอนิเมชันท่องเที่ยวบอกเล่าได้เวลา
ถูกเรียกคืนการพลิกแอนิเมชันสำหรับแถบนำทางด้านล่างและหน้าจอรับ
- **เอฟเฟกต์พลิก 3 มิติ**: การเปลี่ยนหน้ากับการหมุน 180 องศาบนแกน Y
- **หน้า Log/Shop**: ใช้แอนิเมชันพลิก
- **หน้าจอรับ**: ใช้แอนิเมชันพลิกจากปุ่มกล่องจดหมายและการเชื่อมโยงแบบลึก

#### การรวมกันของการออกแบบ UI
การออกแบบ Neumorphic ถูกนำไปใช้กับหน้าจอส่ง รับ และร้านค้า รวมกันทั้งแอป
- **หน้าจอส่ง/รับ**: ใช้เงา Neumorphic กับการ์ดและกล่องโต้ตอบ
- **หน้าจอร้านค้า**: เอฟเฟกต์เรืองแสงบนปุ่มแท็บ การ์ดผลิตภัณฑ์สไตล์ Neumorphic
- **การออกแบบช่องป้อนข้อมูล**: สไตล์ในส่วนแยกชัดเจนพื้นที่ป้อนข้อมูล

#### แอนิเมชันการจัดเรียงลำดับ Log ใหม่
เมื่อรายการย้ายไปยังส่วนบนสุดของรายการ log เนื่องจากความเห็นใหม่ จะใช้แอนิเมชันที่นุ่มนวล
- **เอฟเฟกต์สเกล**: รายการขยายขึ้นชั่วครู่ จากนั้นกลับไปขนาดปกติหลังจากการจัดตำแหน่ง
- **การประมวลผลตามลำดับ**: รายการหลายรายการเคลื่อนไหวตามลำดับเมื่อย้ายพร้อมกัน
- **เน้นอักษร**: ระบุชัดเจนว่ารายการใดเป็นรายการที่สุดท้ายที่สุด

#### การปรับเสถียรภาพของฐานข้อมูล
ปัญหาความเข้ากันได้ของฐานข้อมูลที่อาจเกิดขึ้นระหว่างการอัพเดตแอปได้รับการแก้ไข
- **การกู้คืนอัตโนมัติ**: อัตโนมัติสร้างฐานข้อมูลใหม่เมื่อเกิดความไม่ตรงกันของเวอร์ชัน
- **การรักษาข้อมูล**: ไม่มีการสูญหายข้อมูลด้วยการซิงโครไนซ์เซิร์ฟเวอร์อัตโนมัติ
- **ทุกแพลตฟอร์ม**: ใช้การแก้ไขเดียวกันกับ iOS, Android, macOS และ Windows

#### การปรับปรุงการล็อครหัสผ่าน
ความปลอดภัยและความสามารถในการใช้งานของรายการที่ป้องกันด้วยรหัสผ่านได้รับการปรับปรุงอย่างมาก
- **ออเวอร์เลย์พร่ามัว**: ซ่อนเนื้อหาอย่างสมบูรณ์เมื่อเข้าไปในรายการที่ล็อค เพื่อเพิ่มความปลอดภัย
- **การออกแบบ Neumorphic**: กล่องโต้ตอบการตั้งรหัสผ่าน/ปลดล็อคใช้สไตล์ Neumorphic
- **การแสดงผล 16 หลัก**: วงกลมเล็ก ๆ (รวม 16) แสดงสถานะการป้อนรหัสผ่าน
- **การป้อนข้อมูลสองบรรทัด**: ตั้งรหัสผ่านและยืนยันบนหน้าจอเดียวกัน

#### การปรับปรุงการตั้งค่าการแจ้งเตือนแบบพุช
หน้าจอการตั้งค่าการแจ้งเตือนแบบพุชได้รับการจัดระเบียบใหม่ตามหมวดหมู่เพื่อให้สะดวกในการจัดการ
- **7 หมวดหมู่**: จำแนกตามธุรกรรม Log, วันหมดอายุ, ปฏิกิริยา, คำหลัก, การแชร์ และระบบ
- **การสลับกลุ่ม**: เปิดหรือปิดหมวดหมู่ทั้งหมดในครั้งเดียว
- **ยุบ/ขยาย**: UI แบบ Accordion อนุญาตให้ดูเฉพาะหมวดหมู่ที่จำเป็น
- **การแจ้งเตือนรายละเอียด**: เพิ่มการอนุมัติ/ปฏิเสธธุรกรรม ปฏิกิริยา คำหลัก การแชร์วันหมดอายุ/รายงาน และการแจ้งเตือนระบบ
- **การออกแบบ Neumorphic**: ใช้สไตล์ Neumorphic กับรายการการแจ้งเตือนแต่ละรายการ

#### การปรับปรุงการตั้งค่าเสียง
ปุ่มปิดเสียงขณะนี้มีการออกแบบ Neumorphic เพื่อการแสดงสถานะการกดแบบสัญชาตญาณ
- **เอฟเฟกต์การกด**: ปุ่มปรากฏว่าถูกกดเข้าด้านในเมื่อปิดเสียง
- **เงา Neumorphic**: สถานะเริ่มต้นแสดงเงานูน สถานะที่กดแสดงเงาเว้า

#### การปรับปรุงประสิทธิการโหลด Log
ความเร็วในการโหลด Log ที่แชร์ได้ถูกปรับปรุง
- **การโหลดที่มีแคชก่อน**: Log ที่มองเห็นก่อนหน้านี้จะแสดงในทันทีจากแคช
- **การประหยัดข้อมูล**: ไม่มีคำขอเซิร์ฟเวอร์เมื่อเยี่ยมชมใหม่ ใช้ข้อมูลท้องถิ่น
- **ความเห็นใหม่แบบเรียลไทม์**: รับความเห็นใหม่ในเวลาจริงแม้เมื่อใช้แคช

#### คุณลักษณะการแก้ไขแบดจ์
แก้ไขรูปภาพและคำหลักของแบดจ์ที่แกะสลัก
- **เมนูแก้ไข**: เมนู "แก้ไขแบดจ์" ปรากฏขึ้นเมื่อเลือกแบดจ์ที่แกะสลัก
- **การเปลี่ยนรูปภาพ**: แทนที่รูปภาพแบดจ์ด้วยรูปภาพใหม่
- **การจัดการคำหลัก**: เพิ่ม ลบ หรือแก้ไขคำหลัก 5 ประเภท

#### การปรับปรุงประสิทธิการใช้รูปภาพ
ความเร็วในการโหลดรูปภาพเครือข่ายได้รับการปรับปรุงอย่างมาก
- **แคชท้องถิ่น**: หลังจากโหลด รูปภาพจะบันทึกในท้องถิ่นเพื่อแสดงในทันที
- **การซิงโครไนซ์รูปภาพแบดจ์**: การเปลี่ยนแปลงรูปภาพแบดจ์สะท้อนในทันทีทั่ว log ทั้งหมด

#### การเติมเต็มอัตโนมัติผู้รับล่าสุด
เลือกผู้รับล่าสุดได้อย่างรวดเร็วบนหน้าจอส่ง
- **รายการเติมเต็มอัตโนมัติ**: แบบหล่นลงแสดงผู้รับล่าสุด 30 รายการเมื่อช่องป้อนข้อมูลโฟกัส
- **การเลือกอย่างรวดเร็ว**: ค้นหาโดยอัตโนมัติเมื่อเลือกผู้รับจากรายการ
- **ค้นหาใหม่**: ล้างการเลือกด้วยปุ่ม X เพื่อค้นหาผู้รับคนอื่น
- **การจัดเก็บแบบท้องถิ่น**: ข้อมูลผู้รับบันทึกโดยอัตโนมัติในท้องถิ่นเมื่อการส่งสำเร็จ

#### การปรับปรุงข้อจำกัดการส่ง
แบดจ์ที่มีอุปกรณ์และแสตมป์ที่แชร์ถูกจำกัดไม่ให้ส่ง
- **แบดจ์ที่มีอุปกรณ์**: ถอดแบดจ์เพื่อส่ง
- **แสตมป์ที่แชร์**: เพิกถอนการแชร์เพื่อส่งแสตมป์
- **คำแนะนำโดยละเอียด**: แสดงข้อความเฉพาะที่อธิบายเหตุผลของข้อจำกัด

#### ระบบปฏิกิริยา
ตอบสนองต่อรายการด้วยการแสดงออกทางอารมณ์
- **12 ปฏิกิริยา**: การแสดงออกทางอารมณ์ที่หลากหลายรวมถึง Like, Love, Best, Congrats และอื่น ๆ
- **การสลับปฏิกิริยา**: ยกเลิกโดยคลิกปฏิกิริยาที่เลือกอีกครั้ง
- **บันทึก Log ของระบบ**: บันทึก log ของระบบบันทึกเมื่อเลือกหรือยกเลิกปฏิกิริยา
- **รองรับหลายภาษา**: ชื่อปฏิกิริยาแสดงในภาษาของผู้ใช้

#### การปรับปรุงฟังก์ชั่นการค้นหา

function copyVersionContent(button, event) {
  event.stopPropagation();
  event.preventDefault();

  const details = button.closest('details.changelog-version');
  const versionContent = details.querySelector('.version-content');
  const versionTitle = details.querySelector('.version-title');

  const titleText = versionTitle ? versionTitle.textContent.trim() : '';
  const contentClone = versionContent.cloneNode(true);

  contentClone.querySelectorAll('h4').forEach(h4 => {
    h4.textContent = '\n## ' + h4.textContent.trim() + '\n';
  });
  contentClone.querySelectorAll('li').forEach(li => {
    li.textContent = '- ' + li.textContent.trim();
  });

  const text = '# ' + titleText + '\n' + contentClone.textContent
    .replace(/\n{3,}/g, '\n\n').trim();

  navigator.clipboard.writeText(text).then(() => {
    const copyIcon = button.querySelector('.copy-icon');
    const checkIcon = button.querySelector('.check-icon');
    copyIcon.style.display = 'none';
    checkIcon.style.display = 'block';
    button.classList.add('copied');
    setTimeout(() => {
      copyIcon.style.display = 'block';
      checkIcon.style.display = 'none';
      button.classList.remove('copied');
    }, 2000);
  });
}
</script>

## 📖 사용자 매뉴얼

Inventory 앱의 모든 기능을 상세하게 알아보세요.

---

### 📦 종이(Paper) 시스템

Inventory에서 **종이**는 아이템을 만들 때 사용하는 핵심 재화입니다.

#### 종이란?
- 아이템을 생성할 때 **종이 1장**이 소모됩니다
- 종이가 없으면 새 아이템을 만들 수 없습니다

#### 종이 획득 방법

| 방법 | 획득량 | 비고 |
|------|--------|------|
| 신규 가입 | 10장 | 최초 1회 |
| Google 연동 | +10장 | 계정 연동 시 |
| Apple 연동 | +10장 | 계정 연동 시 |
| 상점 구매 | 10장 | 인앱 결제 |

> 💡 **팁**: 소셜 계정을 연동하면 무료로 종이를 추가로 받을 수 있어요!

---

### 📄 아이템 생성

#### 아이템 생성하기

1. 홈 화면에서 보유 중인 **종이**를 탭합니다
2. 하단에 나타나는 메뉴에서 **"새 아이템 생성"**을 선택합니다
3. 아이템 정보를 입력합니다:
   - 제목 (필수)
   - 내용
   - 이미지 (선택)
   - 개시일자 (선택)
   - 유효기간 (선택)
4. **저장**을 누르면 종이 1장이 소모되고 아이템이 생성됩니다

#### 개시일자와 유효기간

| 설정 | 설명 |
|------|------|
| **개시일자** | 이 날짜가 되기 전까지 내용이 블러 처리됩니다 |
| **유효기간** | 이 날짜가 지나면 내용이 블러 처리되고, 로그 추가가 불가능합니다 |

설정하지 않으면 즉시 공개되고 영구적으로 유지됩니다.

### 🔒 문서 잠금

문서 아이템에 비밀번호를 설정하여 내용을 보호할 수 있습니다.

#### 문서 잠금이란?

잠금이 설정된 문서는 비밀번호를 입력해야만 내용을 볼 수 있습니다.

| 상태 | 설명 |
|------|------|
| **잠금 해제** | 누구나 내용을 볼 수 있음 |
| **잠금 설정** | 비밀번호 입력 시에만 내용 열람 가능 |

#### 잠금 설정하기

1. 홈 화면에서 잠금할 **문서**를 선택합니다 (여러 개 선택 가능)
2. 하단 메뉴에서 **"아이템 잠금"**을 선택합니다
3. 숫자 비밀번호를 입력합니다 (최대 16자리)
4. 동일한 비밀번호를 다시 입력하여 확인합니다
5. 잠금이 완료되면 아이템에 **자물쇠 아이콘**과 **회색조**가 표시됩니다

#### 잠금 해제하기

1. 잠긴 문서를 탭하여 상세 화면에 진입합니다
2. 비밀번호 입력 다이얼로그가 나타납니다
3. 올바른 비밀번호를 입력하면 잠금이 해제됩니다
4. 한번 해제된 잠금은 영구적으로 해제됩니다 (다시 잠그려면 새로 설정)

> ⚠️ **주의**: 비밀번호를 분실하면 복구가 불가능합니다. 잠긴 문서는 **삭제만 가능**합니다.

#### 잠금 특징

- 잠금 비밀번호는 **클라우드에 저장**됩니다
- 아이템을 다른 사람에게 전송해도 잠금 상태가 유지됩니다
- 잠긴 문서는 검색 결과에 표시되지 않습니다
- 잠긴 상태에서도 삭제는 가능합니다

### 🏷️ 뱃지 시스템

**뱃지**는 앱 내에서 사용할 고유한 신원(Identity)입니다.

#### 뱃지란?

뱃지에 도메인명을 각인하면, 앱 내 모든 활동에서 이메일 대신 **각인한 도메인명**으로 표시됩니다.

예: `MyBrand`, `홍길동`, `Company2025`

#### 뱃지 각인하기

1. 상점에서 뱃지를 구매합니다 (미사용 상태)
2. 홈에서 뱃지를 탭 → **"각인"** 메뉴 선택
3. 원하는 도메인명 입력 (최대 32자)
   - 모든 언어의 문자와 숫자 사용 가능
   - 특수문자, 이모지 사용 불가
4. **"도메인 중복 확인"** 버튼으로 사용 가능 여부 확인
5. 선택적으로 128x128 이미지 첨부
6. **"각인하기"** 버튼으로 완료

> ⚠️ **주의**: 한번 각인된 도메인명과 이미지는 **영구적으로 변경할 수 없습니다**. 신중하게 선택하세요!

#### 뱃지 장착/해제

- **장착**: 앱 내 모든 활동에서 각인한 도메인명으로 표시됩니다
- **해제**: 기본 이메일로 다시 표시됩니다
- 여러 뱃지를 보유해도 한 번에 하나만 장착 가능합니다

### 🔖 스탬프 시스템

**스탬프**는 템플릿을 만들어 다른 사람에게 전달할 수 있는 아이템입니다.

#### 스탬프 템플릿 만들기

1. 상점에서 스탬프를 구매합니다 (미사용 상태)
2. 홈에서 스탬프를 탭 → **"새 템플릿 생성"** 메뉴 선택
3. 템플릿 정보 입력:
   - **이미지** (필수)
   - **제목** (최대 32자)
   - **내용** (최대 1024자)
   - **개시일자** (선택) - 이 날짜 전까지 내용 비공개
   - **유효기간** (선택) - 이 날짜 후 로그 추가 불가
   - **로그 모드** - 단독로그 또는 공유로그
4. **"템플릿 생성"** 버튼으로 완료

#### 스탬프 복제하기

활성화된 스탬프(템플릿)와 종이를 함께 선택하면 **"템플릿 복제"** 메뉴가 나타납니다.

- 종이 1장이 소모됩니다
- 동일한 템플릿의 새 아이템이 생성됩니다
- 공유로그 모드라면 원본과 같은 공유 공간을 사용합니다

### ⚡ 빠른 입력 (Quick Input)

로그 작성 시 `+` 버튼을 누르면 특수 정보를 쉽게 입력할 수 있는 **Quick Input** 메뉴가 나타납니다.

#### Quick Input 유형

| 유형 | 설명 | 예시 |
|------|------|------|
| **키워드** | 일반 키:값 형식 | `상태:양호` |
| **날짜 알람** | 날짜와 시간 (D-Day 자동 계산) | `마감:@2024-12-25@` |
| **전화번호** | 국가별 자동 포맷팅 | `연락처:#010-1234-5678#` |
| **위치** | 주소/장소 정보 | `구매처:%서울시 강남구%` |
| **금액** | 통화와 금액 | `가격:$50000KRW$` |

#### 특수 카드 기능

로그에 입력된 특수 정보는 **카드 형태**로 표시되며, **길게 누르면** 추가 기능이 활성화됩니다:

| 카드 유형 | 길게 누르기 동작 |
|----------|-----------------|
| **📅 날짜** | 알람 설정/해제 |
| **📞 전화번호** | 모바일: 바로 전화 걸기, 데스크톱: 복사 |
| **📍 위치** | 모바일: 지도 앱에서 열기, 데스크톱: 복사 |
| **💰 금액** | 클립보드 복사 |

#### 날짜 알람 설정

날짜 카드를 길게 누르면 알람을 설정할 수 있습니다:

1. 날짜 카드를 **길게 누르기**
2. 확인 다이얼로그에서 **설정** 선택
3. 알람이 설정되면 카드 색상이 변경되고 🔔 아이콘이 표시됩니다
4. 다시 길게 누르면 알람을 해제할 수 있습니다

> 💡 **팁**: 날짜 카드에는 자동으로 D-Day가 계산되어 표시됩니다 (예: D-5, D-Day, D+3)

### 📝 로그 시스템

**로그**는 아이템이나 스탬프 템플릿에 기록을 남기는 기능입니다.

#### 로그 모드

| 모드 | 단독로그 | 공유로그 |
|------|----------|----------|
| **저장 위치** | 각자 개인 공간 | 원본 스탬프 기준 공유 공간 |
| **볼 수 있는 로그** | 자신이 작성한 로그만 | 모든 소유자의 로그 |
| **용도 예시** | 개인 일기, 메모 | 공동 작업, 커뮤니티 |
| **원본 삭제 시** | 영향 없음 | 로그 추가 불가 (읽기만 가능) |

#### 로그 형식

로그를 작성할 때 다양한 형식을 사용할 수 있습니다.

**기본 형식:**

| 형식 | 설명 | 예시 |
|------|------|------|
| 일반 텍스트 | 채팅 말풍선으로 표시 | `오늘 점검 완료!` |
| `키:값` | 정보 카드로 표시 (키 1~8자) | `상태:양호` |
| `---` | 구분선으로 표시 | `---` |

같은 키를 여러 번 사용하면 `이전값 → 새값` 형태로 비교 표시됩니다.

**텍스트 스타일링:**

| 형식 | 효과 | 예시 |
|------|------|------|
| `<[텍스트]>` | 메인 색상 카드 (배경 + 텍스트) | `<[중요]>` |
| `[<텍스트>]` | 회색 카드 + 메인 색상 텍스트 | `[<주의>]` |
| `[텍스트]` | 회색 카드 | `[참고]` |
| `<텍스트>` | 메인 색상 텍스트 | `<강조>` |
| `(텍스트)` | 서브 색상 텍스트 | `(부가정보)` |
| `"텍스트"` | **굵게** | `"중요한 내용"` |
| `'텍스트'` | <u>밑줄</u> | `'강조할 부분'` |
| `!텍스트!` | ~~취소선~~ | `!삭제된 내용!` |

> 💡 스타일은 최대 3단계까지 중첩할 수 있습니다. 예: `<["굵은 메인색"]>`

**자동 로그:**

아이템이 전송되면 자동으로 전송 로그가 기록됩니다:
- `보낸사람 → 받은사람` 형태로 표시
- 뱃지를 장착한 경우 도메인명으로 표시

#### 공유로그 중단 조건

- **원본 스탬프 삭제**: 공유로그 추가 불가 (읽기만 가능)
- **유효기간 만료**: 공유로그 추가 불가 (읽기만 가능)

### 🎨 잉크 시스템

**잉크**는 앱의 테마 색상을 변경하는 아이템입니다.

#### 사용 방법

1. 잉크를 탭하여 상세 화면 진입
2. **"테마 변경"** 버튼 클릭
3. 즉시 앱 전체 색상이 변경됩니다

#### 잉크 종류

| 잉크 | 색상 |
|------|------|
| **블랙 잉크** | 기본 검은색 테마 |
| **골드 잉크** | 고급스러운 금색 테마 |

### 🔍 검색 기능

#### 기본 검색

상단 검색창을 탭하면 검색 모드가 활성화됩니다.

- 검색어를 입력하면 **실시간**으로 결과가 필터링됩니다
- 검색 모드를 종료하려면 X 버튼을 탭합니다

#### 최근 검색어

- 최대 5개의 최근 검색어가 저장됩니다
- 최근 검색어를 탭하면 검색창에 자동 입력됩니다
- X 버튼으로 개별 삭제 가능합니다

#### 필터로 저장

검색어를 **"필터로 지정"** 버튼으로 저장하면:

- 검색창 아래에 필터 태그로 표시됩니다
- 태그를 탭하면 해당 검색어로 즉시 필터링됩니다
- X 버튼으로 필터를 삭제할 수 있습니다

#### 키워드 집계 기능

필터가 설정된 상태에서 아이템을 선택하면, 선택한 아이템들의 **키워드 값을 자동으로 집계**합니다.

**숫자 값 집계:**
- 동일한 키워드의 숫자 값들을 **합산**하고 **평균**을 계산합니다
- 예: `가격:1000`, `가격:2000`, `가격:3000` 선택 시
- 결과: `합계: 6000  평균: 2000`

**문자열 값 집계:**
- 동일한 키워드의 문자열 값들을 **공백으로 연결**합니다
- 예: `태그:A`, `태그:B`, `태그:C` 선택 시
- 결과: `A B C`

**값 복사하기:**
- 집계 결과 카드를 **길게 누르면** 값이 클립보드에 복사됩니다
- 복사 완료 시 알림이 표시됩니다

> 💡 **활용 예시**: 여러 아이템의 가격 합계를 빠르게 계산하거나, 관련 태그들을 한 번에 복사할 수 있습니다.

### 📤 아이템 보내기

중고 거래나 선물 시 아이템을 함께 전송할 수 있습니다.

#### 보내기 순서

1. 하단 탭에서 **보내기** 선택
2. 전송할 아이템을 **체크**합니다 (여러 개 선택 가능)
3. 받는 사람 이메일을 입력합니다
4. **보내기** 버튼 클릭

#### QR 코드로 이메일 입력

상대방 이메일을 직접 입력하는 대신 QR 코드를 활용할 수 있습니다:

1. 받는 사람이 자신의 **프로필** 화면에서 QR 코드를 표시합니다
2. 보내는 사람이 **QR 스캔** 버튼을 누릅니다
3. 카메라로 QR 코드를 스캔합니다
4. 이메일이 자동으로 입력됩니다

> ⚠️ **중요**: 전송한 아이템은 내 인벤토리에서 **즉시 사라집니다**. 전송 후에는 취소할 수 없습니다.

---

### 📥 아이템 받기

다른 사람이 보낸 아이템을 받을 수 있습니다.

#### 받기 순서

1. 하단 탭에서 **받기** 선택
2. 받은 아이템 목록이 표시됩니다
3. 각 아이템에 대해:
   - **수락**: 내 인벤토리에 추가됩니다
   - **반송**: 보낸 사람에게 돌려보냅니다

#### 알림

- 새 아이템이 도착하면 푸시 알림을 받습니다 (알림 허용 시)
- 받기 탭에 미수락 아이템 개수가 표시됩니다

### 🛒 상점 이용하기

상점에서 다양한 아이템을 구매할 수 있습니다.

#### 구매 가능 상품

| 상품 | 용도 |
|------|------|
| **종이** | 아이템 생성용 (소모품) |
| **스탬프** | 템플릿 생성용 |
| **뱃지** | 고유 도메인 각인용 |
| **잉크** | 테마 색상 변경용 |

#### 구매 방법

1. 하단 탭에서 **상점** 선택
2. 원하는 상품을 탭합니다
3. 결제를 진행합니다 (App Store / Google Play)
4. 결제 완료 후 자동으로 지급됩니다

> 💡 모든 플랫폼(iOS, Android, Windows, macOS)에서 동일하게 상점을 이용할 수 있습니다.

### 📂 파티션 기능

아이템이 많아지면 파티션으로 그룹화할 수 있습니다.

#### 파티션 만들기

1. 아이템을 선택합니다
2. 하단 메뉴에서 **"파티션 생성"** 선택
3. 파티션 이름을 입력합니다 (최대 16자)

#### 파티션 특징

- 파티션은 **기기에만 저장**됩니다
- 다른 기기에서는 파티션 구조가 보이지 않습니다
- 아이템 자체는 클라우드에 저장되어 모든 기기에서 볼 수 있습니다
